# atvcompanion.in

A single static HTML page for the bare-domain landing of
**atvcompanion.in** — the origin story for Vikram Devatha's Astrology
101 companion ("the blue file"). Hosted on Hostinger.

## What's here
- `index.html` — the page (origin story · how it came to be)
- `assets/blue-file.jpg` — the photo
- `CLAUDE.md` — Claude Code reference for this repo
- `LICENSE`

## Deploy
Hostinger Git-syncs `main` into `public_html/`. A push to `main` is a
deploy. No build step, no server, no dependencies.

## The companion app
The full Express + Supabase companion app lives in a separate repo:
[sreebalakrishnan/app.atvcompanion.in](https://github.com/sreebalakrishnan/app.atvcompanion.in)
— deployed at [app.atvcompanion.in](https://app.atvcompanion.in).
