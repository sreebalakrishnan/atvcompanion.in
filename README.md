# atvcompanion.in — landing branch

A single static HTML landing page for **atvcompanion.in**, hosted on Hostinger.

This is the `landing` branch. It contains only:

- `index.html` — the origin story (How it came to be · the blue file)
- `assets/blue-file.jpg` — the photo
- `LICENSE`

Hostinger Git-syncs this branch into `public_html/`. No build step, no server, no dependencies.

The full Express + Supabase companion app lives on `main` and is deployed separately (Node host required).
