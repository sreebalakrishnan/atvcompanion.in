# Deck 64: The Houses

**Source file:** `64. The Houses.pdf`
**Type:** Astrology fundamentals / Houses

## Topic
The first deep-dive into the 12 Houses, listing their classical significations from multiple Sanskrit texts. Opens with a "12 matchboxes" exercise to physically anchor each House to personal symbolic objects. This is version 1 of "The Houses" (iterated as decks 67, 70, 74).

## Key concepts
- 12 Matchboxes exercise: label 12 matchboxes 1-12, find tiny symbolic objects for each House, deposit them, discuss in class
- Each House is described from multiple classical sources: BPHS, Uttara Kalamrita, Sarvartha Chintamani, Hora Sara, Brihat Jataka
- 1H (Tanu Bhava) — Kendra, Trikona, Dharma; karaka Sun. Body, appearance, intellect, complexion, vigor, weakness, happiness/grief, innate nature, longevity
- 2H (Kutumbha Bhava) — Maraka, Artha, Panapara; karaka Jupiter. Wealth, grains, family, speech, faith, eyes, valuables

## Sanskrit/technical terms introduced
- Tanu Bhava — body house (1H)
- Kutumbha Bhava — family house (2H)
- Kendra — angle (1, 4, 7, 10)
- Trikona — trine (1, 5, 9)
- Maraka — killer house (2, 7)
- Panapara — succedent (2, 5, 8, 11)
- Apoklima — cadent (3, 6, 9, 12)
- Dharma, Artha, Kama, Moksha — Purushartha groupings

## Names / references mentioned
- Parashara — Brihat Parashara Hora Shastra (BPHS)
- Kalidasa — Uttara Kalamrita
- Varahamihira — Brihat Jataka
- Prithuyasas — Hora Sara
- Venkata Sharma — Sarvartha Chintamani

## Notes
Deck only reaches 1H and 2H in the extracted text — appears truncated relative to later versions (which cover all 12). Successive versions (67, 70, 74) progressively cover more Houses with the same template.
