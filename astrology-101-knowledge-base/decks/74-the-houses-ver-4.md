# Deck 74: The Houses (ver 4)

**Source file:** `74. The Houses ver 4.pdf`
**Type:** Astrology fundamentals / Houses

## Topic
Fourth iteration of "The Houses" deck. Same template and content as version 2 (deck 67). Extracted text covers Houses 1-8.

## Key concepts
- 12 Matchboxes exercise (carried over)
- For each House: BPHS Sanskrit shloka + transliteration + English translation, then significations from Uttara Kalamrita, Sarvartha Chintamani, Hora Sara
- Houses covered in extracted text:
  - 1H Tanu Bhava — Kendra, Trikona, Dharma; Sun
  - 2H Dhana Bhava — Maraka, Artha, Panapara; Jupiter
  - 3H Sahaja Bhava — Apoklima, Upachaya, Kama; Mars
  - 4H Sukha Bhava — Kendra, Moksha; Moon
  - 5H Putra Bhava — Panapara, Trikona, Dharma; Jupiter
  - 6H Ripu Bhava — Apoklima, Upachaya, Dushtana, Artha; Mars and Saturn
  - 7H Yuvati Bhava — Kendra, Kama, Maraka; Venus and Jupiter
  - 8H Randhra Bhava — Dushtana, Panapara, Moksha; Saturn (BPHS shloka adds: Randhra signifies longevity, death, destruction, hidden parts, poverty, defects)

## Sanskrit/technical terms introduced
- Same Sanskrit House names as decks 67/70

## Names / references mentioned
- Same classical sources: Parashara (BPHS Ch 11), Kalidasa (Uttara Kalamrita), Varahamihira (Brihat Jataka), Prithuyasas (Hora Sara), Venkata Sharma (Sarvartha Chintamani)

Largely identical to version 2 (deck 67). Goes one House further than version 3 (deck 70) in extracted text (reaches 8H vs 7H), but does not exceed deck 67's coverage of all 12. Likely the latest working draft as of class delivery; the four versions appear to be successive editing snapshots rather than substantively different decks.
