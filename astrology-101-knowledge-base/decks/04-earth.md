# Deck 4: Earth

**Source file:** `4. Earth.txt`
**Type:** Astronomy fundamentals

## Topic
First astronomy deck — establishes Earth as reference frame for astrology. Introduces degrees, geographic grid, Earth's motions, and the chain of astronomical concepts that lead to astrological constructs.

## Key concepts
- Course outline / "Wait, but why?" list: Degrees -> Earth size & shape -> Latitudes/equator -> Longitudes/prime meridian -> Rotation, time zones, time difference per degree -> Tilt -> Ecliptic -> Revolution -> Ecliptic vs Equator -> Length of day/night by latitude -> Solstices -> Equinoxes -> Seasons. Each ties to an astrology concept: Ascendant, Transits, Ayanamsha, Rahu/Ketu, Tropical vs Sidereal Astrology.
- Degrees: 360 in a circle; 1 degree = 60 minutes (60'); 1 minute = 60 seconds (60''). Different from clock hh:mm:ss.
- Rigveda 1.164.48 quoted: "Twelve spokes, one wheel, navels three... three hundred and sixty like pegs."
- Earth: circumference 40k km, diameter 12-13k km, 70% water
- Latitude/longitude as a grid; tilt; revolution; tropics

## Sanskrit/technical terms introduced
- Ayanamsha — alluded to (precession-related offset)
- Rahu / Ketu — alluded to (lunar nodes)

## Names / references mentioned

- Sources: timeanddate.com, scienceabc.com, Wikipedia
