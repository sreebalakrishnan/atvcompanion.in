# Deck 6: How to Learn

**Source file:** `6. How to learn.txt`
**Type:** Logistics / Pedagogy

## Topic
Short meta-deck setting expectations and learning posture for the program. Three pillars: online classes, retreats, self work.

## Key concepts
- Don't try to remember details — look for themes that come up in your life. Treat stories as mirrors.
- Contradictions are okay; many versions and interpretations exist (like viewing the same tree from different directions).
- Pick a thread, go deep, make it personal — see where it leads.
- Learn in layers — revisit topics (e.g., Saturn) and you'll see something new each time.
- This subject does not give neat closure; sit with knowing or not-knowing — that's where the real magic happens.

## Notes
Brief deck. Establishes a non-linear, reflective, layered learning approach — important framing for the mythology and self-work portions of the curriculum.
