## Topic
Introduces the mentorship program structure: small fortnightly peer groups of 7-8, organized around named mentors, run on Microsoft Teams.

## Key concepts
- Mentor sessions exist to help students think, reflect and speak about key topics.
- Mentors ask questions; there is no teaching; no right or wrong answers.
- Sessions are not recorded.
- Small circles of 7-8 per group; same group meets regularly.
- Fortnightly cadence (every two weeks), starting August 2025.
- Held on Microsoft Teams; timings are in IST.
- Students select a time slot via WhatsApp poll; weekday options span Sunday through Saturday.
