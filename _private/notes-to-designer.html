Libre Caslon Condensed ⚠ Substituted

Brand spec: Libre Caslon Condensed for all major headings (H1).

What we use: Libre Caslon Display as primary, Libre Caslon Text as fallback.

Why: Libre Caslon Condensed is not available on Google Fonts. Only two variants are freely available on the web: Libre Caslon Display (designed for large headline sizes) and Libre Caslon Text (optimised for smaller sizes). We use Display as the closest match to the condensed headline intent.

Font stack currently in use across all pages:

'Libre Caslon Display', 'Libre Caslon Text', 'Cormorant Garamond', serif

To close this gap: If you have a licensed WOFF2 file for Libre Caslon Condensed, we can self-host it and replace Display entirely — making the implementation fully spec-compliant.

What was not implemented

Logo / Brandmark

The ATV logo/brandmark does not appear in the app. This is a cohort-only learning tool, not a public marketing surface — it uses the "Astrology 101 Companion" product identity. If you'd like the logo added anywhere (header, about page, footer), please provide SVG assets and we'll integrate them.

Color misuse rules

The guidelines specify avoiding gradients, pure black/white pairings, and mid-tone-on-mid-tone combinations. The app follows all of these — no gradients, no pure black backgrounds, and accent colours always paired with light backgrounds.

Full typography hierarchy enforcement

The brand specifies a strict H1 → H2 → Body → H3/CTA hierarchy. The app generally follows this but pages were built incrementally — not every page explicitly enforces Switzer Medium for H2 subtitles. This is a planned cleanup item.

Questions — decisions needed from designer

  • Libre Caslon Condensed font files — can you share WOFF2 files so we can self-host? This would close the one remaining type gap and make headlines fully spec-compliant.
  • Logo integration — should the ATV brandmark appear anywhere in the app (header, about page, footer)? If yes, please share an SVG file.
  • Background tone check — the app now uses Sunlit White (#F8F6EF) as the primary background. Does this read correctly against your brand materials? The previous value (#F4EEDA) was slightly warmer and more golden — happy to revert if preferred.
